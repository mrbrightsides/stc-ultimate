## Technical Questions

### What is HCPS (Human Cyber-Physical Systems)?

**HCPS** is a four-layer architectural framework that integrates:

1. **Physical Layer** üèóÔ∏è
   - Real-world tourism assets (hotels, temples, beaches)
   - IoT sensors (temperature, crowd density)
   - GPS location services
   - RFID/NFC check-in systems

2. **Cyber Layer** üíª
   - Blockchain ledger (payments, contracts)
   - AI/ML models (recommendations, predictions)
   - 3D virtual environments (metaverse)
   - Cloud databases (off-chain data)

3. **Collaborative Layer** ü§ù
   - Multi-user interactions (group planning)
   - DAO governance (community voting)
   - Real-time chat and coordination
   - Shared itineraries

4. **Human Layer** üë•
   - Tourists, SME operators, researchers, admins
   - User interfaces and dashboards
   - Decision-making and feedback
   - Social interactions

**Why HCPS?**
- Seamless integration of physical and digital worlds
- Human-centric design (people, not just technology)
- Collaborative decision-making (not top-down)
- Holistic view of tourism ecosystem

**Academic Background:**
- Extends Industry 4.0 concepts to Tourism 5.0
- Based on IEEE research in smart cities
- Published frameworks in tourism management journals

### What is Tourism 5.0?

**Tourism 5.0** is the next evolution of tourism:

| Era | Characteristics | Example |
|-----|----------------|---------|
| **Tourism 1.0** | Mass tourism, package tours | 1950s-1970s bus tours |
| **Tourism 2.0** | Online booking, DIY planning | Booking.com, Tripadvisor |
| **Tourism 3.0** | Mobile apps, social media | Instagram travel, Airbnb |
| **Tourism 4.0** | Smart tourism, IoT, data-driven | Smart cities, AI assistants |
| **Tourism 5.0** | Human-centric, sustainable, collaborative | **STC Ultimate** |

**Five Pillars of Tourism 5.0:**

1. **Human-Centricity**
   - Technology serves human needs (not vice versa)
   - Personalized experiences
   - Accessible to all (virtual tours for disabled)

2. **Sustainability**
   - Environmental monitoring (carbon footprint tracking)
   - Overtourism prevention (crowd alerts)
   - Local community benefits (fair revenue distribution)

3. **Resilience**
   - Prepared for disruptions (pandemics, disasters)
   - Digital alternatives (virtual tourism during lockdowns)
   - Decentralized (no single point of failure)

4. **Transparency**
   - Blockchain-verified transactions
   - Open data for research
   - Clear pricing (no hidden fees)

5. **Collaboration**
   - Multi-stakeholder participation
   - DAO governance
   - Shared decision-making

**STC Ultimate implements all five pillars.**

## Troubleshooting

### I can't connect my wallet

**Common Fixes:**

1. **MetaMask Not Installed**
   - Install from https://metamask.io/
   - Refresh page after installation

2. **Wrong Network**
   - Switch to Sepolia Testnet
   - In MetaMask: Networks ‚Üí Sepolia

3. **Browser Issues**
   - Use Chrome, Firefox, or Brave
   - Disable other wallet extensions (Phantom, Coinbase Wallet)
   - Clear browser cache

4. **Mobile Connection**
   - Use MetaMask mobile app browser
   - Or WalletConnect (tap "Connect" ‚Üí "WalletConnect")

**Still Not Working?**
- Check MetaMask is unlocked
- Try private/incognito window
- Contact support with screenshot

---

### The app says "Transaction Failed"

**Checklist:**

1. **Check Balance**
   - Do you have enough ETH for payment + gas?
   - Get free Sepolia ETH: https://sepoliafaucet.com/

2. **Check Network**
   - Are you on Sepolia Testnet?
   - App should auto-prompt to switch

3. **Check Gas Limit**
   - Usually auto-set correctly
   - Try increasing manually in MetaMask (Advanced)

4. **Check Contract Address**
   - Verify correct contract (app shows address)
   - Don't approve suspicious contracts

5. **Try Again**
   - Sometimes network congestion causes failures
   - Wait 1 minute and retry

**If Still Failing:**
- Copy error message
- Contact support with:
  - Wallet address
  - Transaction hash (if available)
  - Screenshot of error

---

### I don't see my booking after payment

**Possible Causes:**

1. **Transaction Still Pending**
   - Check MetaMask "Activity" tab
   - Wait for confirmation (30-60 seconds)
   - Refresh page after confirmation

2. **Wrong Wallet Connected**
   - Make sure same wallet that paid is connected
   - Check address in app vs MetaMask

3. **Browser Cache Issue**
   - Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
   - Clear cache and cookies
   - Re-connect wallet

4. **Transaction Reverted**
   - Check Etherscan for transaction status
   - If reverted, funds returned to wallet
   - Try booking again

**Need Help?**
- Check Transaction History in app
- Search your transaction on Etherscan: https://sepolia.etherscan.io/
- Contact support with transaction hash

---

### The virtual tour won't load

**Troubleshooting:**

1. **Internet Connection**
   - Virtual tours are large (100-500MB)
   - Use WiFi, not mobile data
   - Check connection speed: https://fast.com/

2. **Browser Compatibility**
   - Use Chrome or Firefox (recommended)
   - Safari may have WebGL issues
   - Update browser to latest version

3. **Device Performance**
   - Close other tabs/apps
   - 3D tours need good GPU
   - Mobile devices may lag on complex tours

4. **WebGL Support**
   - Check: https://get.webgl.org/
   - Enable hardware acceleration in browser settings
   - Update graphics drivers

**Alternative:**
- Try 360¬∞ panorama view (lighter)
- Use lower quality setting (if available)
- Try different device
