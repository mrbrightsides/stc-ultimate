## 📹 **Recording Tips**

1. **Camera Setup**: Face camera when introducing, share screen for demo
2. **Pacing**: Speak slowly and clearly — don't rush through features
3. **Cursor Movement**: Move cursor smoothly, pause on key UI elements
4. **Annotations**: Use screen annotation tools to highlight important areas
5. **Retakes**: If you make a mistake, pause, then restart from that scene
6. **Background Music**: Optional light background music (keep volume low)
7. **Captions**: Consider adding captions for accessibility
